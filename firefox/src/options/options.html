How to get an Atlos API token:

  1. Navigate to the project for which you need an API token.
  2. Navigate to the Access page of the project.
  3. Press the Create button in the API Tokens section of the page.
  4. Assign the token a name, write a description of what the token is used for, and select all three permissions: Read, Comment, and Edit for the token.
  5. Click the Create Token button.
  6. Paste token above along with the project name.
